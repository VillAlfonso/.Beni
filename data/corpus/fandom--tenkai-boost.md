Tenkai Boost has multiple missions for one to complete. This is a list of missions you'll expect to see and of course do if you want to see more.
- Buy a power up! (0/1)
- Collect 50 blocks in one run! (0/50)
- Collect a power up! (0/1)

- Reach 250M in one run! (0/250)
- Reach 150M without collecting coins! (0/150)
- Reach 500M without power ups! (0/500)

## List of Power Ups

The game contains power ups, which can be used to help one advance themselves farther in the run. Here's a list of power ups accessible in the game.
- Speed: It looks like Bravenwolf's back unit (Tenkai Booster). With this power up, you can move very fast and pass through any obstacle without being hit. If you want to buy this power up from the shop, it will cost 300 core bricks.
- Shield: It looks like Bravenwolf's shield. With this power up, you can survive from an attack. If you want to buy this power up from the shop, it will cost 300 core bricks.
- Sonic Blast: It looks like Tributon's bow. With this power up, you can destroy any obstacle that's coming your way. If you want to buy this power up from the shop, it will cost 300 core bricks.
- Revive: It looks like Bravenwolf's chestplate. With this power up, you can be given a second chance after being hit by an obstacle to get back on the run. If you want to buy this power up from the shop, it will cost 600 core bricks.
